# **App Name**: StartupOS

## Core Features:

- Startup Profile Management: Capture and store essential startup information, including industry and stage, to build a foundational data record.
- Financial Health Tracking: Monitor and display key financial metrics such as cash runway and revenue within a centralized interface for real-time visibility.
- Team Management: Organize and manage team member profiles, roles, and contributions for improved coordination.
- Project & Task Management: Organize and track ongoing projects and tasks with status updates, fostering efficient execution.
- AI Growth Intelligence Engine: A generative AI tool providing intelligent insights for strategic decision-making and sustainable growth, based on collected startup data.
- Central Analytics Dashboard: Provide a high-level overview of critical KPIs and metrics from all integrated modules on a centralized dashboard.

## Style Guidelines:

- Primary color: A sophisticated Navy (#0F172A) to convey authority and stability, suitable for key UI elements and text against a light background.
- Background color: A clean and neutral off-white (#F8FAFC) as the primary surface color, enhancing readability and providing a spacious feel.
- Accent color: An Electric Blue (#3B82F6) for interactive elements, highlights, and calls to action, symbolizing action and technology.
- Primary typeface: 'Inter' (sans-serif) for all text. Chosen for its modern, clean, and objective aesthetic, ensuring excellent readability across headlines and body content.
- Use 'Lucide-React' for icons, providing a consistent, modern set of vector icons that align with a professional SaaS interface and 'Command Center' theme.
- The UI will feature a persistent sidebar navigation and a dynamic content area, designed to evoke a 'Command Center' feel. It will include dashboard overview cards for key performance indicators (KPIs) and will be fully responsive across devices.
- Subtle transition animations for route changes and state updates will enhance the user experience without distracting from the data-focused nature of the platform.
